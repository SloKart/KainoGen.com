How to check our work

We are proud of our ability to maximize the performance that Google uses to decide which websites to link to customers.

Go to Google's Online

and enter the URL of any website you'd like to test

If you prefer to run the test yourself:

  • from a desktop
    Download Google Chrome

    If you already are using Google Chrome you can skip this step. otherwise you can download it here

  • go to webpage you want to audit

    The homepage is usually the best place to start.

  • right click

    and select inspect.

  • click "lighthouse"

    it is usually in the upper right corner and may be behind a drop down arrow.

  • Make sure you are on the right page

    you can still edit the web address at the top of the screen to go to the home page.

  • click "Analyze Page Load"

    in a few moments, you will see how good google thinks a website is.
